## Deskripsi Singkat

Program ini merupakan aplikasi berbasis website 3D hollow object viewer yang dapat menampilkan beberapa model-model objek 3 dimensi yang dibuat menggunakan WebGL. Dalam menampilkan model, program juga dapat:

- Mengubah model yang ditampilkan
- Menggunakan *shading* pada model
- Mengubah warna *lighting* model, serta mengubah radius dan melakukan rotasi pada *lighting* tersebut
- Menampilkan model dengan beberapa jenis proyeksi (*orthographic*, *perspective*, *oblique*)
- Melakukan pengubahan radius serta rotasi kamera terhadap sumbu X dan Y
- Transformasi objek yang terdiri dari dilatasi (*scale*) serta translasi dan rotasi yang dapat dilakukan terhadap sumbu X, Y, dan Z.

Selain model yang tersedia di dalam website, model juga dapat di-load oleh pengguna dari file eksternal bertipe JSON dengan format tertentu.

Program ini dibuat untuk memenuhi Tugas Besar 2 mata kuliah IF3260 Grafika Komputer.

## Cara Menjalankan Program

Program dapat dijalankan dengan membuka file `src/index.html` pada browser.
